斗地主 - 网页版 + WebAssembly 内核 + AI 辅助
一个可直接在浏览器运行的完整斗地主游戏，前端使用原生 HTML/CSS/JavaScript，核心游戏规则与本地 AI 逻辑由 C 语言实现并编译为 WebAssembly，同时支持接入大模型实现出牌提示、对局复盘、AI 决策等增强能力。
✨ 项目亮点
开箱即用：纯网页单页应用，无需安装，打开浏览器即可游玩
高性能内核：斗地主发牌、抢地主、牌型判断、出牌规则、胜负判定全部在 C 层实现
多难度本地 AI：内置简单 / 普通 / 困难三档电脑出牌逻辑
双模式对战：支持「本地 AI」和「大模型人机 AI」两种电脑玩家模式
完整大模型能力
局中实时 AI 出牌提示
局后自动 AI 复盘分析
支持对话式 继续追问
电脑玩家可由大模型 全程决策
完整游戏体验：自带牌面素材、全套音效、流畅动画与交互
🎮 游戏流程
开始游戏 → 2. 抢地主 → 3. 玩家与两名电脑轮流出牌
局中随时使用 AI 提示
结束后查看 AI 复盘
🧩 页面功能
游戏状态与当前回合展示
玩家身份、积分、剩余牌数
实时牌型提示
手牌区、出牌区、结算区完整布局
AI 配置面板（API、模型、Key）
音效开关、显示设置
🏗️ 技术架构
前端
index.html：主界面、交互逻辑、AI 调用、音效、渲染
game.js：WebAssembly 胶水代码
game.wasm：C 编译后的游戏内核
C / WebAssembly 核心
1.c：斗地主规则、游戏流程、对外接口
ai_logic.c / ai_logic.h：本地 AI 出牌与抢地主策略
game_shared.h：公共结构体、枚举声明
大模型接口
qwen_client.c / qwen_client.h：兼容 OpenAI 格式的 C 端模型调用封装
qwen_example_usage.c：对局分析与复盘示例
资源
牌组/：扑克牌图片资源
.mp3：发牌、出牌、炸弹、王炸、胜负等音效
