
# 斗地主

一个基于 `C + WebAssembly + HTML/CSS/JavaScript` 实现的网页版斗地主项目。  
项目包含完整的发牌、抢地主、出牌、过牌、AI 自动出牌、积分结算、音效播放，以及可选的 AI 提示与局后复盘功能。

适合用于：

- WebAssembly 游戏练手项目
- 前端 + C 混合架构示例
- 斗地主规则与出牌流程演示
- 带 AI 提示能力的轻量网页游戏

---

## 项目特色

- 基于 `C` 编写核心游戏逻辑
- 通过 `Emscripten` 编译为 `WASM` 在浏览器运行
- 网页端可直接游玩，无需安装客户端
- 支持抢地主、出牌、过牌、自动轮转
- 支持 AI 玩家自动决策
- 支持积分统计与结算展示
- 支持音效、背景音乐、牌面图片切换
- 支持局中 AI 提示与局后 AI 分析
- 支持继续追问 AI，围绕当前对局做交互式解释

## 4. 页面交互

网页端提供了完整的操作界面，包括：

- 开始游戏
- 出牌
- 过牌
- 刷新界面
- 配置面板
- 日志查看
- 局中 AI 提示
- 局后 AI 分析

同时支持：

- 牌面图片显示
- 背景音乐
- 出牌音效
- 日志面板展开/收起
- 结果面板展示

---

## 5. AI 辅助功能

除了内置的本地 AI 出牌逻辑，页面还支持接入外部大模型，提供：

### 局中 AI 提示
可解释当前局势，并给出：
- 推荐动作
- 推荐原因
- 是否建议保留炸弹
- 当前局势偏保守还是进攻

### 局后 AI 分析
对整局进行复盘，输出：
- 关键转折点
- 错失最优出牌
- 高风险操作
- 胜负原因总结

### AI 继续追问
用户可以在 AI 面板中继续追问，例如：
- 为什么推荐这手？
- 还有没有别的可选方案？
- 如果想保留对子该怎么出？

## 运行方式

### 1. 编译 WASM

使用 `Emscripten` 编译：

```powershell
emcc 1.c ai_logic.c -o game.js ^
  -s WASM=1 ^
  -s MODULARIZE=1 ^
  -s EXPORT_NAME=GameModule ^
  -s EXPORTED_FUNCTIONS="['_main','_malloc','_free','_game_init','_game_play','_game_pass','_game_ai_step','_game_auto_run','_game_get_state_json','_rob_landlord','_ai_rob_landlord']" ^
  -s EXPORTED_RUNTIME_METHODS="['UTF8ToString','HEAP32']"
```

---

### 2. 启动静态页面

编译完成后，使用任意静态服务器打开项目目录，例如：

```powershell
python -m http.server 8000
```

然后在浏览器访问：

```text
http://localhost:8000
```
